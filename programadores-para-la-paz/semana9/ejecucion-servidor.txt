Al ejecutar el comando node server.js, Node.js inicia la aplicación y pone el servidor en funcionamiento.

En la terminal aparece el siguiente mensaje:
Servidor ejecutándose en puerto 3000

Al visitar la ruta http://localhost:3000 se muestra el mensaje:
Servidor activo y funcionando

Cuando se visita la ruta http://localhost:3000/estado, el servidor responde con un objeto JSON.
Respuesta obtenida:

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
